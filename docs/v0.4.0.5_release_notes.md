# v0.4.0.5 AnalysisScreen UI Extraction

This release does not add a new user-facing feature.

It extracts analysis detail UI sections into a smaller composable file while
keeping screen-level state, destination selection, and ViewModel wiring unchanged.

Key points:
- `AnalysisScreen.kt` remains the owner of screen-level destination state and ViewModel callbacks.
- Strength trend cards are split into focused section composables.
- The shared analysis chart section UI is moved with those extracted sections.
- Analysis calculations, metric registry, metadata, backup, record, and plan behavior are unchanged.
- Chart data sources and series construction inputs are unchanged.
- ViewModel, Repository, analysis engine, Room schema, and public APIs are unchanged.
