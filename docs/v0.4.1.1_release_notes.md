# v0.4.1.1 Smash Speed Service Extraction Release Notes

v0.4.1.1 is a behavior-preserving repository extraction release.

It moves smash speed read/write responsibilities out of `TrainingRepository` without changing smash speed behavior, Lab metrics, backup, record, metadata, analysis, plan, home, readiness, or UI behavior.

Changes:

- Added `SmashSpeedService` for date-scoped observe/add/delete operations.
- Kept `TrainingRepository` public APIs stable by delegating existing smash speed calls to the new service.
- Preserved existing attempt index assignment, validation, DAO call order, and date key behavior.
- Added focused service tests for add/delete behavior.
- No Room schema change.

Korean summary:

v0.4.1.1은 `TrainingRepository`의 smash speed 조회/추가/삭제 책임을 별도 service로 분리한 구조 개선 릴리스다.

smash speed 동작, Lab metric, 백업, 기록, 메타데이터, 분석, 계획, 홈, readiness, UI 동작은 변경하지 않는다.
