# v0.4.1.7 Backup Restore Importer Extraction Release Notes

v0.4.1.7 is a test-first backup restore importer extraction release.
It adds focused restore behavior regression coverage before moving restore CSV import logic out of `TrainingRepository`.
It does not change backup format, restore semantics, metadata resolution, analysis, UI, Room schema, or ViewModel call sites.

## Changes

- Added `BackupRestoreImportBehaviorTest`.
- Added `BackupRestoreImportService`.
- Delegated restore CSV import orchestration from `TrainingRepository` to the new service.
- Kept `TrainingRepository.importRecordsBackup(...)` public behavior unchanged.
- Kept daily-timeseries import behavior unchanged.
- Bumped app version to `v0.4.1.7` / `401007`.

## Restore Behavior Coverage

- Runtime metadata override restore and override precedence.
- Custom exercise stableKey and runtime override restore.
- DailyMetric sleep as canonical CheckIn sleep.
- CheckIn sleep promotion when DailyMetric is missing.
- Smash speed duplicate skip.
- Workout entry/set grouping, set order, confirmed/unconfirmed state, and duplicate entry skip.
