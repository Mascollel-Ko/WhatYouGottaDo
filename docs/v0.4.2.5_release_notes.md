# v0.4.2.5 Release Notes

v0.4.2.5 is a fatigue/readiness wording hotfix.

## Cause

- v0.4.2.4 correctly separated OFI overall status from axis warnings, but several user-facing Korean strings still needed final wording alignment.

## Changes

- Renamed the Home current line to `현재 상태: <score> · <OFI label>`.
- Prefixed OFI display labels with `피로도` where appropriate, such as `피로도 보통` and `피로도 낮음`.
- Restored canonical axis display names `관절/건/충격` and `회복 지속`.
- Updated very-high, high, and all-good axis guidance sentences to the requested wording.
- Bumped the app version to v0.4.2.5 / 402005.

## Not Changed

- OFI classifier logic.
- Fatigue axis calculation.
- Readiness thresholds.
- Projected/expected fatigue logic.
- Plan, record, backup, metadata, and program behavior.

## Tests

- Focused `TodayFatigueStatusLabelerTest` and `HomeFatigueCardSummaryFactoryTest`.
- Full `testDebugUnitTest`.
