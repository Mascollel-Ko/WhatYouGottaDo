# v0.4.1.8 Backup Restore Importer Extraction Completion Release Notes

v0.4.1.8 finishes the backup restore importer extraction started in v0.4.1.7.
TrainingRepository.importRecordsBackup now delegates restore CSV import behavior to BackupRestoreImportService.
The private TrainingRepository importRestoreCsv body was removed.
Backup format, restore semantics, metadata resolution, analysis, UI, Room schema, and ViewModel call sites are unchanged.

## Korean Summary

v0.4.1.8은 v0.4.1.7에서 시작한 backup restore importer service 분리를 완료한 릴리스다.
TrainingRepository.importRecordsBackup은 BackupRestoreImportService를 실제 production flow에서 사용하며, TrainingRepository 내부의 private importRestoreCsv 본체는 제거된 상태다.
백업 포맷, 복원 동작, metadata resolution, 분석, UI, Room schema, ViewModel call site는 변경하지 않는다.

## Verification Note

The v0.4.1.7 baseline already contained the required production wiring and private importer removal when v0.4.1.8 work began. This release records that completion, updates version metadata, and preserves the existing tested behavior.
