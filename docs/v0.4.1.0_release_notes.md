# v0.4.1.0 Program Planning Service Extraction Release Notes

v0.4.1.0 is a behavior-preserving repository extraction release.

It moves program planning / program persistence / program apply orchestration responsibilities out of `TrainingRepository` without changing ProgramBuilder, generated program results, plan UI, record, metadata, backup, analysis, readiness, or home behavior.

Changes:

- Added `ProgramPlanService` for program observation, item observation, create/save/delete, item mutation, conflict summary, and apply-to-dates orchestration.
- Kept `TrainingRepository` public APIs stable by delegating existing program calls to the new service.
- Left `generateProgramSkeleton`, `ProgramSkeletonGenerator`, ProgramBuilder validation, plan UI, record mutation, metadata resolution, backup/import/export, analysis, readiness, and home behavior unchanged.
- No Room schema change.

Korean summary:

v0.4.1.0은 `TrainingRepository`의 프로그램 저장/적용/삭제/조회 조립 책임 일부를 별도 service로 분리한 구조 개선 릴리스다.

`ProgramBuilder`, 자동 프로그램 생성 결과, 계획 UI, 기록, 메타데이터, 백업, 분석, readiness, 홈 동작은 변경하지 않는다.
