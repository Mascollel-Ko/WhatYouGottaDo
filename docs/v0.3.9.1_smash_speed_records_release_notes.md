# v0.3.9.1 Smash Speed Records Release Notes

v0.3.9.1 adds first-class badminton smash speed records.

## Changes

- Added date-scoped manual smash speed attempts in km/h.
- Added per-day summary values: best speed, average speed, top-3 average speed, and attempt count.
- Added Room persistence for `smash_speed_records` with migration 16 -> 17.
- Added restore CSV export/import support with `row_type=smash_speed`.
- Added Lab metrics for smash best speed, average speed, top-3 average speed, and attempt count.
- Added a compact Record screen card for entering, listing, and deleting smash speed attempts.

## Notes

- Missing days remain missing/null in analysis series; they are not filled as zero.
- Existing workout, fatigue, badminton transfer, and program builder formulas are unchanged.
- Existing v0.3.9.0 tag is not modified.
