# v0.3.6.5 Cleanup Release Notes

## Scope

v0.3.6.5 is a cleanup and stability release. It does not add a new user-facing training feature.

## Changes

- Aligns the app release identity to `versionName = "v0.3.6.5"` and `versionCode = 306005`.
- Aligns the canonical metadata manifest app-version fields with the current app release.
- Keeps the canonical metadata taxonomy revision and metadata artifact version unchanged.
- Keeps Room database version `16` and existing schema exports unchanged.
- Prepares the repository for a separate outputs/document artifact cleanup commit.

## Verification Notes

This release note does not claim a full local debug build. The cleanup task intentionally avoids repeated local `assembleDebug` runs. Use GitHub Actions if configured, or run the smallest relevant local checks manually.
