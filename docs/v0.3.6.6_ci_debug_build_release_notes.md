# v0.3.6.6 CI Debug Build Release Notes

v0.3.6.6 is an infrastructure and stability release. It does not add a new user-facing training feature.

## Changes

- Adds a GitHub Actions Android debug build workflow.
- Runs `testDebugUnitTest` remotely on pushes, pull requests, tags, and manual dispatch.
- Runs `assembleDebug` remotely after unit tests.
- Uploads the generated debug APK as a workflow artifact named `WhatYouGottaDo-debug-apk`.
- Aligns the app release identity to `versionName = "v0.3.6.6"` and `versionCode = 306006`.

## Scope

- No training logic changes.
- No fatigue or analysis logic changes.
- No metadata taxonomy changes beyond current app-version manifest fields.
- No ProgramBuilder changes.
- No UI changes.
- No Room database schema change.

## Verification

Final debug build verification is expected to run through GitHub Actions. Local `assembleDebug` was not run for this infrastructure release to avoid local Gradle network and permission loops.
