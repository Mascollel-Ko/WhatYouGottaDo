# v0.4.1.6 Metadata Editor Extraction Release Notes

v0.4.1.6 is a behavior-preserving metadata editor extraction release.
It first adds focused regression tests for exercise metadata editor, runtime metadata override, and custom exercise deletion behavior, then extracts that responsibility from `TrainingRepository` into `ExerciseMetadataEditorService`.

## Changes

- Added focused regression tests for exercise metadata editor behavior.
- Added `ExerciseMetadataEditorService`.
- Delegated existing `TrainingRepository` metadata editor public APIs to the new service.
- Preserved public APIs, ViewModel/UI call sites, seed/bootstrap behavior, backup/restore behavior, metadata resolution semantics, analysis, fatigue, badminton transfer, Lab, home, readiness, and program behavior.
- Bumped app version to `v0.4.1.6` / `401006`.

## Test Coverage

- New exercise editor defaults and copy source sorting.
- Existing seed exercise effective metadata editor data.
- Custom exercise creation with unique stable key and runtime metadata override persistence.
- Existing exercise update with stable key preservation.
- Editor validation without partial writes.
- Seed metadata override reset behavior.
- Missing exercise reset fallback.
- Runtime metadata override priority.
- Full resolved metadata map coverage.
- Active/archive toggling without deleting overrides.
- Unused custom exercise deletion safety.
