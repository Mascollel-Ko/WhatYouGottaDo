# v0.4.2.2 Release Notes

v0.4.2.2 is a readiness threshold relaxation hotfix.

## Cause

- Readiness presentation and residual pressure gates were still slightly too conservative.
- OFI label thresholds should stay stable while readiness restrictions and volume reduction guidance trigger a little later.

## Changes

- Kept OFI thresholds unchanged at 75 / 87 / 98.
- Raised presentation score gates from 69 / 81 to 72 / 84.
- Raised presentation volume gates from 52 / 69 / 87 to 54 / 72 / 90.
- Raised residual pressure ratio gates from 1.3225 / 1.5525 / 1.84 to 1.3754 / 1.6146 / 1.9136.
- Raised residual z-score gates from 1.15 / 1.725 / 2.30 to 1.196 / 1.794 / 2.392.
- Bumped the app version to v0.4.2.2 / 402002.

## Not Changed

- OFI label thresholds.
- Fatigue/load calculation formulas.
- Graph integration.
- Court duration, smash speed, time-series, or strength analyses.
- UI structure.

- `./gradlew.bat testDebugUnitTest`
- `./gradlew.bat assembleDebug`
