# v0.4.1.10 ProgramBuilder Policy Extraction Release Notes

v0.4.1.10 is a test-first ProgramBuilder policy extraction release.
It adds focused prescription, duration, and session-fit regression coverage before moving those policies out of ProgramBuilder.
It does not change candidate selection, generated program behavior, fatigue gate semantics, template selection, validation, UI, repository behavior, Room schema, metadata, backup, or analysis behavior.

## Korean Summary

후보 선택, 생성 프로그램 동작, fatigue gate 의미, template selection, validation, UI, repository, Room schema, metadata, backup, analysis 동작은 변경하지 않는다.

## Changes

- Added focused regression coverage for ProgramBuilder prescription policy behavior.
- Extracted `ProgramPrescriptionPolicy` for prescription calculation, exercise count, warmup reserve, duration estimate, and required-slot session-fit trimming.
- Kept ProgramBuilder candidate filtering, scoring, slot selection, fatigue gating, warnings, validation, and output model behavior unchanged.
