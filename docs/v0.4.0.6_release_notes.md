# v0.4.0.6 Repository Extraction

This release does not add a new user-facing feature.

v0.4.0.6 is a behavior-preserving repository extraction release. It moves
performance trend / read-only analysis summary orchestration out of
`TrainingRepository` without changing analysis calculations, metadata
resolution, muscle-load series, backup, record, plan, or UI behavior.

Key points:
- `TrainingRepository.performanceTrendSummary()` remains the public entry point.
- `PerformanceTrendSummaryService` now owns performance trend summary assembly.
- Metric series builder call order and inputs are unchanged.
- Resolved runtime metadata catalog wiring is preserved.
- ViewModel, UI, backup, Room schema, and analysis formulas are unchanged.

## Korean Summary

v0.4.0.6은 `TrainingRepository`의 performance trend / read-only analysis summary 조립 책임 일부를 별도 service로 분리한 구조 개선 릴리스다.

분석 계산, metadata resolution, muscle-load series, 백업, 기록, 계획, UI 동작은 변경하지 않는다.
