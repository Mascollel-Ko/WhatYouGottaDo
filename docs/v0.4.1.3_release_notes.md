# v0.4.1.3 Analysis Summary Service Extraction Release Notes

v0.4.1.3 is a behavior-preserving repository extraction release.
It moves read-only fatigue and badminton transfer analysis summary assembly out of `TrainingRepository` without changing analysis calculations, metadata resolution, date-window semantics, Lab metrics, backup, record, calendar, program, home, readiness, or UI behavior.

## Changes

- Added `AnalysisSummaryService`.
- Delegated `fatigueAnalysisHistory`, `badmintonTransferSummary`, and `badmintonTransferCoverageSummary` from `TrainingRepository`.
- Kept `TrainingRepository` public APIs and ViewModel/UI call sites unchanged.
- Left calculation engines, metric registries, metadata resolver behavior, backup, record, calendar, program, home, readiness, and UI paths untouched.
