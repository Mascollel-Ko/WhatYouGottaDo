# v0.5.0.0 Release Notes

v0.5.0.0 is the first major quiet UI refinement release. It makes the existing product easier to scan without changing training, OFI, connective-tissue, strength, badminton or ProgramBuilder calculations.

## Baseline

- Started from latest `origin/main` at `7105d96d2209d30e0c1bb8c6a97a5c8e3b318243`.
- Previous app identity: `v0.4.2.16 / 402016`.
- Baseline debug APK: `46,198,651` bytes.

## Cause

- Bottom navigation still used temporary `H/R/P/E/A` marks.
- Home and Analysis relied on repeated filled surfaces and card-inside-card patterns.
- The OFI overall status and canonical five-axis state were separated, so the default view required more scanning than necessary.
- Similar selectors, spacing and emphasis used slightly different treatments across major screens.

## Changes

### Bottom navigation

- Replaced alphabet marks with one outlined Material icon family.
- Mapping: Home/home, Record/edit note, Plan/event note, Exercise/fitness center, Analysis/analytics.
- Korean labels remain visible. One restrained indicator and brand color identify the selected tab; unselected tabs remain neutral.

### Surface hierarchy

- Home summary metrics are text columns rather than filled pills inside a card.
- Analysis hub entry points are two section cards containing rows and dividers, replacing six independent large cards.
- Connective-tissue child rows, fatigue detail rows, recognition groups and expanded performance details no longer create card-like surfaces inside their parent card.
- Shared information messages are unframed helper text instead of another filled card.

### OFI summary

- The current OFI status is followed by the canonical five axes in a quiet text list:
  `고중량·힘 신경계`, `전신 근육`, `국소 근육`, `고속`, `반응`.
- Low and ordinary states remain neutral. Only high and very-high states receive strong warning emphasis.
- No axis-specific rainbow colors, large bars, repeated icons or contributor dumps were added.
- Existing contributor detail remains behind the existing detail action.

### Shared visual system

- Light and dark themes now use neutral background/surface colors with restrained blue, green and amber accents.
- Material typography weights and a 4/6/8/12/16dp shape scale are defined centrally.
- Major screens use 20dp horizontal insets and 16dp section spacing.
- Analysis and fatigue selectors share one neutral Material `FilterChip` implementation.
- Home gives one light brand tint to the primary record action; secondary entry points remain neutral.
