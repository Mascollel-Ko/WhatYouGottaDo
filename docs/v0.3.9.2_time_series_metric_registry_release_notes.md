# v0.3.9.2 Time Series Metric Registry Release Notes

v0.3.9.2 expands the Lab/time-series metric layer used for relationship analysis and future IRF work.

## Added

- Squat e1RM Lab metric.
- Deadlift e1RM Lab metric.
- Muscle-group load metrics for 15 buckets.
- Daily, recent 3-day, and recent 7-day windows for every muscle-group load bucket.
- Lab metric category display support for performance, muscle load, and smash speed categories.

## Notes

- This release does not implement Bayesian IRF, regression estimation, or new chart engines.
- No Room schema change.
- No backup schema change.
- No exercise seed or canonical metadata taxonomy rewrite.
