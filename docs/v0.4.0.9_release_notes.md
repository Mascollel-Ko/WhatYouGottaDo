# v0.4.0.9 Home Summary Service Extraction Release Notes

v0.4.0.9 is a behavior-preserving repository extraction release.

Changes:

- Added `HomeSummaryService` for Home today summary assembly.
- Added `CoachingSignalsSummaryService` for coaching signals summary input assembly.
- Added `TodayStatusSummaryService` for today readiness and phase-aware today status wrappers.
- Kept `TrainingRepository` public APIs stable by delegating to the new services.
- Preserved readiness, projected fatigue, daily check-in, record, metadata, backup, analysis, plan, and UI behavior.
