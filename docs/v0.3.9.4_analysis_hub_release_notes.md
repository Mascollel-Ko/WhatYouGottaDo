# v0.3.9.4 Analysis Hub Release Notes

## Summary

v0.3.9.4 restructures the Analysis tab into a hub that opens focused analysis screens instead of showing every detailed block in one long feed.

## Changes

- Reworked the Analysis main screen into three entry cards:
  - 오늘 컨디션 및 피로도 분석
  - 배드민턴 전이 분석
  - 근력운동 추이 분석
- Kept the existing training distribution summary on the Analysis hub.
- Combined today condition, fatigue axis status, and fatigue cause context into the fatigue detail flow.
- Moved the detailed fatigue analysis and weekly OFI trend into the fatigue detail screen.
- Moved badminton transfer coverage and badminton training-load charts into the badminton detail screen.
- Removed the less reliable badminton transfer type share and top exercise stimulus chart options from the user-facing badminton detail card.
- Added strength trend detail content for:
  - bench press, squat, and deadlift e1RM trend
  - muscle-group load share
  - muscle-group load share trend
  - rep-range share
  - rep-range share trend

## Notes

- e1RM and rep-range analysis use confirmed performed sets only.
- Missing e1RM weeks are not filled with zero.
- Badminton weekly charts are weekly point charts, not monthly aggregation.
- No Room schema change.
