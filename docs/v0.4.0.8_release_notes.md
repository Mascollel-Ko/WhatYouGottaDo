# v0.4.0.8 Daily Service Extraction Release Notes

v0.4.0.8 is a behavior-preserving repository extraction release.

Changes:

- Added `DailyStatusService` for DailyMetric and DailyCheckIn read/write orchestration.
- Added `DailyReadinessInputService` for today readiness input assembly.
- Kept `TrainingRepository` public APIs stable by delegating to the new services.
- Preserved canonical `sleepHours` behavior through `DailyMetric.sleepHours`.
- Left backup, record mutation, metadata, analysis, plan, and UI behavior unchanged.

Korean summary:

v0.4.0.8은 `TrainingRepository`의 DailyMetric / DailyCheckIn / daily readiness input 관련 책임 일부를 별도 service로 분리한 구조 개선 릴리스다.

수면, 일일 체크인, readiness, 운동 후 예상 피로도, 백업, 기록, 메타데이터, 분석, 계획, UI 동작은 변경하지 않는다.
