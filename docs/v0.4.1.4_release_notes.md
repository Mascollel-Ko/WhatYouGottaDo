# v0.4.1.4 Read Query and Program Generation Service Extraction Release Notes

v0.4.1.4 is a behavior-preserving repository extraction release.
It moves simple read-only query delegation and program generation input assembly out of `TrainingRepository` without changing public APIs, ViewModel/UI call sites, generated program behavior, `ProgramSkeletonGenerator`, `ProgramBuilder`, `ProgramBuilderValidation`, metadata resolution, fatigue calculation, backup, record, calendar, program persistence, analysis, home, readiness, or UI behavior.

## Changes

- Added `RepositoryReadQueryService`.
- Added `ProgramGenerationService`.
- Kept the two services independent.
- Kept `TrainingRepository` public APIs and ViewModel/UI call sites unchanged.
- Left `ProgramSkeletonGenerator`, `ProgramBuilder`, `ProgramBuilderValidation`, `ProgramPlanService`, metadata resolver behavior, fatigue calculation, backup, record, calendar, home, readiness, analysis, and UI paths untouched.
