# v0.4.2.4 Release Notes

v0.4.2.4 is a fatigue/readiness display separation hotfix.

## Cause

- The Home "현재 피로도 상태" card displayed an OFI score with a label derived from axis warnings.
- A single very-high axis could make the overall current status look like "피로 심화" even when the canonical OFI value was normal.

## Changes

- Kept the overall current status label based only on the canonical OFI classifier.
- Added a separate current-axis warning message for high and very-high fatigue axes.
- Added a level-count line for current axes: very high, high, normal, and low.
- Reused the same formatter in Home and analysis readiness UI paths.
- Preserved projected/expected fatigue wording separately from current-axis guidance.
- Bumped the app version to v0.4.2.4 / 402004.

## Not Changed

- OFI calculation.
- Fatigue axis calculation.
- Readiness thresholds.
- Expected/projected fatigue calculation.
- Plan, record, backup, metadata, and program logic.
