# v0.4.0.2 Backup Structure Refactor

This release does not add a new user-facing feature.

It extracts backup export/import responsibilities out of `TrainingRepository`
into smaller backup services and isolates exercise metadata override backup
mapping.

Key points:
- Existing backup CSV behavior is preserved.
- Exercise metadata overrides remain included in backup export/import roundtrip.
- Seed metadata remains the fallback for built-in exercises without overrides.
- Restored metadata still resolves as seed plus override, with override taking precedence.
- v0.4.0.1 metadata and muscle-load behavior is preserved.
