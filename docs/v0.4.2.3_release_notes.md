# v0.4.2.3 Release Notes

v0.4.2.3 is a fatigue/readiness display wording hotfix.

## Cause

- OFI fatigue, readiness caution axes, and weekly residual pressure could be displayed with names that made them look like the same fatigue index.
- The readiness thresholds already match the requested relaxed v0.4.2.2 values and remain unchanged in this patch.

## Changes

- Kept OFI as the current fatigue and fatigue-change display source.
- Renamed readiness status wording to "주의할 피로 축".
- Renamed readiness restriction lists to "오늘 가능한 운동" and "오늘 조절할 운동".
- Renamed projected fatigue wording to "끝나면 예상 피로도".
- Renamed weekly residual/pressure composite wording to "이번 주 누적 부담".
- Bumped the app version to v0.4.2.3 / 402003.

## Thresholds

- OFI thresholds remain unchanged: 75 / 87 / 98.
- Readiness presentation, volume, pressure ratio, and z-score gates remain at the relaxed v0.4.2.2 values.

## Not Changed

- Fatigue/load calculation formulas.
- Residual decay or aggregation formulas.
- Graph integration behavior.
- Court duration, smash speed, time-series, or strength analysis models.
- UI structure.

## Tests

- `./gradlew.bat testDebugUnitTest`
- `./gradlew.bat assembleDebug`
