# v0.3.4.5 Change Summary

## Version

- `versionName`: `v0.3.4.5`
- `versionCode`: `304045`
- Room DB version: `12`
- Exercise seed version: `6`

## Main Changes

| Area | Change | Notes |
| --- | --- | --- |
| Record tab | Added exercise-level delete action | Removes the full workout entry from today's record with confirmation when needed. |
| Record tab | Fixed last-set deletion edge case | An unconfirmed entry can now be fully removed instead of becoming undeletable after sets are deleted. |
| Rest overlay | Suppressed overlay when no next target exists | Overlay now requires an active timer, a same-day target, a next target, and a confirmed-set origin. |
| Performance trend UI | Shortened section title and renamed metrics | `성과 추세`, `RPE 대비 운동량`, and `셔틀 플레이 시간` are used. |
| Performance trend UI | Simplified detail selector behavior | Metric chips appear only in trend mode; incompatible chart combinations remain blocked. |
| Performance trend UI | Removed unsuitable ranking modes | Strength and fatigue ranking are hidden; badminton ranking remains. |
| Relationship analysis | Replaced large chip rows with dropdowns | X/Y selection is compact and prevents same-metric scatter plots. |
| Badminton metrics | Split shuttle play from drills/support | Direct play uses structured metadata and stable exercise keys. |
| Seed data | Added `배드민턴 레슨` | Seed version bumped to `6`. |
