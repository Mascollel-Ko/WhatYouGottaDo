# v0.3.4.4.1 Change Summary

## 버전

- `versionCode`: 304041
- `versionName`: 0.3.4.4.1

## 주요 변경

- 초기 프로필 입력을 숫자/선택/multi-select 중심으로 재구성했다.
- DB v10과 `MIGRATION_9_10`을 추가했다.
- 구조화 profile 필드를 CSV 백업/복원에 포함했다.
- cold-start readiness 보정이 구조화 필드만 사용하도록 수정했다.
- Room schema 10을 생성했다.
- migration test를 v9->v10 기준으로 추가했다.

## 하지 않은 일

- 새 분석 차트 추가 없음
- 고급 분석 구현 없음
- 자동 프로그램 생성 고도화 없음
- 기록 탭 compact UI 재설계 없음
- 오버레이 권한 정책 변경 없음

## 남은 리스크

- v0.3.4.4에서 이미 저장된 복잡한 자유 텍스트는 완전 변환하지 않는다.
- legacy 문자열은 가능한 범위에서만 구조화하고, 실패 시 null/default/OTHER로 둔다.
