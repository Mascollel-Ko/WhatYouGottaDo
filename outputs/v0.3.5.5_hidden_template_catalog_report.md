# v0.3.5.5 Hidden Template Catalog Report

## Scope

v0.3.5.5 adds the first representative hidden-template catalog on top of the v0.3.5.4 policy-table architecture. It does not implement the complete ratio x goal x days matrix.

Implemented representative templates:

| Template ID | Days | Weeks | Badminton:strength | Focus |

| `FOOTWORK_COD_4D_4W_80_20` | 4 | 4 | 80:20 | Footwork/COD |

| `STRENGTH_BIASED_5D_8W_30_70` | 5 | 8 | 30:70 | Strength-biased |

All other combinations continue through the existing policy-based day-count fallback. No implicit full matrix was added.

## Architecture

- `ProgramTemplateCatalog.select(request)` owns representative-template matching.
- Each representative template defines ordered `ProgramSlotId` requirements, exercise role, and required-anchor status.
- `ProgramBuilder` asks the catalog for a selection and fills its slots; it does not contain a ratio/days/weeks template `if`/`when` matrix.
- `SlotCapabilityResolver` supplies metadata-first capability profiles.
- `CoverageAccountingPolicy` determines whether a candidate can fill the requested template slot and how strongly it fits.
- `ExposureTargetTable` supplies duration/day-adjusted numeric targets used by the scoring balance adjustment.
- `FatigueSlotPolicy` remains the safety gate and adapts planned day intensity.
- Non-representative combinations retain the previous `roles()`-based fallback behavior.

## Anchor Preservation

Every representative template marks weekly squat, hinge/posterior-chain, and upper-pull exposures as required anchors. Required anchors:

- are ordered before optional trimming can remove them;
- are reduced to fewer sets before exceeding the session budget;
- produce a HARD `TEMPLATE_REQUIRED_SLOT_UNFILLED` or `TEMPLATE_REQUIRED_SLOT_OVERRUN` validation issue when they cannot be safely fulfilled;
- cannot be filled by rehab-like activation candidates.

The safety policy still has precedence. If fatigue or available metadata makes an anchor unsafe or unavailable, generation reports a hard issue rather than silently substituting unrelated work.
