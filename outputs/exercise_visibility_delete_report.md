# Exercise Visibility / Delete Report

## 구현

- 운동 탭 관리 모드에서 숨김, 숨김 해제, 삭제를 제공한다.
- 숨김 운동은 기본 운동 선택/검색 화면에서 제외된다.
- 숨김 보기 필터로 다시 확인하고 복구할 수 있다.

## 삭제 정책

- 사용자 추가 운동이고 어디에서도 참조되지 않을 때만 실제 삭제한다.
- 기록 또는 프로그램에서 참조 중이면 삭제하지 않는다.
- 기본 제공 운동은 참조가 없어도 삭제하지 않고 숨김을 사용하도록 유도한다.

## 검사한 관계

- `WorkoutEntry.exerciseId`
- `TrainingProgramItem.exerciseId`

## 남은 리스크

- 분석/추천은 현재 DB의 운동 목록과 기록을 기반으로 동작한다. 별도 분석 캐시가 생기면 삭제 전 캐시 참조도 검사해야 한다.
