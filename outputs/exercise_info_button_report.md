# Exercise Info Button Report

## 변경

- `ExerciseInfoDialog`를 공통 UI로 추가했다.
- 운동 탭과 기록 탭이 같은 운동 정보 dialog를 사용한다.
- 기록 탭 운동 카드 헤더의 운동명 옆에 `i` 버튼을 추가했다.

## 표시 정보

- 운동 이미지
- 운동명
- 카테고리 / detail
- 설명
- 기본 휴식
- 주동근, 보조근, 장비, 패턴, 분류, 축부하, 측성, 훈련역할, 전이 정보, metadata confidence

## 제한

- 없는 메타데이터는 추정하지 않는다.
- 운동별 과거 최고 기록/평균 기록/추세는 v0.3.4.5 후보로 보류했다.
