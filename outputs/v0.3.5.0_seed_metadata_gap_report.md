## Report Columns

- exerciseName: seed exercise display name
- stableKey: stable seed key
- currentCategoryType: current category, movement category, force type, and movement pattern
- estimatedMovementFamily: report-only family estimate from structured seed metadata
- missingDetailedMuscleTag: future detailed tag template expected by family
- missingMuscleContribution: future contribution template expected by family
- missingProgramSlot: future program slot expected by family
- missingRedundancyGroup: future redundancy group expected by family
- suspiciousSportSessionProgramSelectable: sport-session rows that must not become program-selectable
- badmintonDirectPlayConflict: badminton direct-play/session classification review
- highAxialPlyometricDecelerationConflict: high-stress family review flags for future session-level validator
- sourceIdIssue: missing, deprecated, unknown, or network-recheck source notes
- automaticFixPossible: whether template defaults could be applied later
- manualReviewRequired: whether a human should review before seed mutation

## Notes

- C_HORIZONTAL_ROW_EMG_NEEDS_SOURCE is treated as deprecated. Row variants should use C_INVERTED_ROW_EMG_SNARR_ESCO_2013 or another verified/candidate row source.
- needs_network_recheck sources are allowed in planning reports but should not justify high-confidence automatic seed updates alone.
- Muscle contribution templates are internal heuristics, not exact physiological claims.
- This step intentionally does not run a full app build because no app code or schema changed.
